### Features:

- Registration & Authorization
- Chats management (add / rename / remove)
- Push notifications
- Words filtering

#### Used technologies:

```
- Javascript
- React.js
- Redux TK
- Bootstrap
- Axios
- Formik
- i18n
- Lodash
- Leo-profanity
- Rollbar
- Socket.io
- Yup
- @hexlet/chat-server (backend)
```

#### Setup

```
  git clone https://github.com/worldspawn-web/frontend-project-12
  make install
  make start
```
